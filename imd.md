## Списки
Чтобы создать неномерной список небходимо начать элемент списка с (*)

Например:

* элемент списка 1
* элемент списка 2
* элемент списка 3

для того чтобы создать номерной список, необходимо начать элемент списка с порядковой цифры

Например:

1. Прервый элемент
2. Второй элемент
3. Третий элемент
